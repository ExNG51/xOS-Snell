# Snell 管理脚本

> 纯自用。

## 默认
```bash
wget -O snell.sh --no-check-certificate https://git.io/Snell.sh && chmod +x snell.sh && ./snell.sh
```

```bash
wget -O snell.sh --no-check-certificate https://gitee.com/ten/Snell/raw/master/Snell.sh && chmod +x snell.sh && ./snell.sh
```

## 注意
* 请手动放行防火墙相应端口。
